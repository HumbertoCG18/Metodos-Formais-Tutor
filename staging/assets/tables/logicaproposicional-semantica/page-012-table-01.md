| Classificação das Fórmulas
• Uma fórmula A é uma tautologia (ou válida) se e somente
se toda interpretação I:→B para A é modelo
• Ou seja, para toda interpretação I, [A] =V
I
• Nesse caso escrevemos ⊨ 𝐴
• Uma fórmula A é uma contradição (ou inválida) se e
somente se nenhuma interpretação I:→B para A é
modelo
• Ou seja, para toda interpretação I, [A] =F
I
• Nesse caso escrevemos ⊭ 𝐴
• Uma fórmula A é satisfatível se e somente se existe ao
menos uma interpretação I:→B para A que é modelo
• Ou seja, existe pelo menos uma interpretação I, [A] = V
