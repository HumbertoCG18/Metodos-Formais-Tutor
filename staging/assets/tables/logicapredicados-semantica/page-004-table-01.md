| Interpretação
• Os elementos básicos de uma interpretação para uma
assinatura da lógica de predicados são:
• Um domínio, o qual é uma coleção não vazia de elementos, no
qual as fórmulas serão avaliadas.
• Um elemento do domínio, para cada constante da assinatura.
• Um conjunto de funções sobre o domínio, uma para cada símbolo
de função presente na assinatura.
• Um conjunto de relações sobre o domínio, uma para cada símbolo
de predicado na assinatura. |
