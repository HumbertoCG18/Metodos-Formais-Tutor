## **Formalização de algoritmos como equações recursivas – Parte 1** 

- 1) Nos exercícios abaixo, salvo restrição específica, o agente computacional é capaz de executar apenas as operações indicadas. 

   - a. Descreva via equações uma função para somar dois números naturais. Use apenas operações de soma/subtração de uma unidade. 

   - b. Descreva via equações uma função para calcular o fatorial de um número natural. A utilização de operação de multiplicação é permitida. 

   - c. Descreva via equações uma função para calcular a potência de dois ( 2[𝑛] ) para _n_ um número natural. A utilização de operação de multiplicação é permitida. 

   - d. Descreva via equações uma função para calcular o n-ésimo termo da série de Fibonacci. 

   - e. Descreva via equações uma função de divisão inteira entre números naturais usando o método das subtrações sucessivas. 

- 2) Analise a sequência de operações abaixo e defina uma função via equações recursivas otimizada para a multiplicação de dois números (a otimização em relação ao exemplo disponível no resumo). 

|5 *3|=|5 * 3 + 0||4 * 4|=|4 * 4 + 0|

||=|2 * 6 + 3|||=|2 * 8 + 0|
||=|1 * 12 + 3|||=|1 * 16 + 0|
||=|0 * 24 + 15|||=|0 * 32 + 16|
||=|15|||=|16|

- 3) Com relação à função para computar o fatorial ( 𝑓𝑎𝑡: ℕ→ℕ ) no exercício 1, elabore uma versão complementar com recursão na cauda ou não dependendo da versão já criada. 

- 4) Complete as funções propostas nos exercícios anteriores especificando para cada uma as pré e pós condições correspondentes caso não tenha já realizado esta tarefa.
